This repository contains the original (legacy) version of the **Lucaflix** backend.

## 📌 Contexto / Context

Este proyecto fue desarrollado en una etapa inicial de aprendizaje.  
En ese momento tenía poca experiencia en buenas prácticas como Clean Code, pruebas automatizadas y manejo avanzado de errores, y solo conocía arquitectura en capas básica (Controller → Service → Repository).  
This project was developed in an early learning stage.  
At that time, I had little experience with best practices like Clean Code, automated testing, and advanced error handling, and only knew basic layered architecture (Controller → Service → Repository).
